JS variables

  1. var: The var keyword was used in all JavaScript code from 1995 to 2015. This keyword should only be used in code written for older browsers.
    • Only use var if you MUST support old browsers.
  2. const: The const keywords were added to JavaScript in 2015.
    • always use const if the value or type should not be changed.
    • const are not hoisted
  3. let: The let keywords were added to JavaScript in 2015.
    • Only use let if you can't use const.
    • Variables declared with let have Block Scope
    • Variables declared with let must be Declared before use
    • Variables declared with let cannot be Redeclared in the same scope
    • let const are not hoisted. Meaning: You can use the variable before it is declared.
